# 🏠 Real Estate Management System - CLI Version

### By: Itay Vazana

This project is a **command-line interface (CLI)** software for managing a real estate agency, written in **Java** as part of my **Object-Oriented Programming (OOP)** course.

---

## 📌 Project Overview

This system was developed as part of an **introductory OOP course** during my early days of learning software development.  
The project applies core **object-oriented programming principles**, including:
- ✅ Encapsulation
- ✅ Inheritance
- ✅ Polymorphism

The software provides basic management features for a small real estate agency, offering functionalities such as property and customer handling, rent calculation, and more.

---

## ✨ Features

- 🏢 Add and remove apartments
- 📑 Create independent copies of specific apartments
- 📋 Organize customers into lists based on their interest in properties
- 🔎 Search and calculate rent by apartment address
- 📊 Additional property management utilities

---

## 🔧 Technologies Used

- Language: **Java**
- Paradigm: **Object-Oriented Programming (OOP)**
- Interface: **Command Line Interface (CLI)**
