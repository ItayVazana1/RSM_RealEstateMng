Itay Vazana

CLI-style real estate agency management software.

Written in JAVA and based on object-oriented programming principles.
The software allows :

- Adding and removing apartments
- Creating an independent copy for a certain apartment
- Organization of customers into lists according to interest in the apartment
- Rent calculator (the search is carried out by the address of the apartment)
- and more...

*** A GUI version will be uploaded soon ***

The software was written as a final project in an object-oriented programming course
